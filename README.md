🚀 Modern Coding Academy – Landing Page
HTML / CSS / JavaScript を使って、ゼロから作り上げたランディングページです。
このプロジェクトは、
「デザインの世界観を形にする」
「UI/UX を自分の手で作り込む」
という目標で、試行錯誤しながら完成させました。

✨ こだわったポイント
🎨 デザインの世界観を統一
- ガラスモーフィズム
- 深海 × エネルギーコアのような光の表現
- グラデーションの質感
- 余白と視線誘導のバランス
細かい色味や影の強さまで調整して、
「見た瞬間に気持ちいい」デザインを目指しました。

📱 レスポンシブ対応
スマホ・タブレット・PC すべてで快適に見られるように、
レイアウトを丁寧に調整しています。

🍔 ハンバーガーメニューの実装
- transform を使ったスライドアニメーション
- pointer-events の制御
- 固定ヘッダーとの干渉を解消
- ページ内リンクが正しく動くように調整
何度もバグと戦いながら、
「気持ちよく開閉できるメニュー」を実現しました。

🔗 ページ内リンクの最適化
固定ヘッダーに隠れないように
padding-top や id の付け方を工夫し、
スムーズに目的のセクションへ移動できるようにしています。

🧩 使用技術
- HTML5
- CSS3（Flexbox / グラデーション / ガラスモーフィズム）
- JavaScript（ハンバーガーメニュー開閉）
├── index.html
├── style.css
└── script.js
